# ledjay-skills

Agent skills by ledjay (Jérémie Gisserot) — reusable capabilities for AI coding agents.

## Installation

```bash
npx skills add jeremiegisserot/ledjay-skills
```

These skills work with:
- Letta Code
- OpenAI Codex
- Claude Code (via conversion)
- Cursor
- GitHub Copilot
